# Monorepo-template

## Requirements

- [NodeJs](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/getting-started/install)

## Usage

- ### Install the dependencies

      yarn

- ### Build

      yarn build

  For a specific packge

      yarn build-scope [package_name]

- ### Watch (for development)

      yarn watch

  For a specific packge

      yarn watch-scope [package_name]
